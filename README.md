# WasteTiresPyrolysis
This repository contains the source code for the techno-economic, life cycle, and safety analyses for the pyrolysis of waste tires as discussed in [[1]](#1). 

## Getting started

A tutorial on how to rerun the analyses can be found the Pyrolisis_Biosteam_Jupyter.ipynb notebook. In order to run the code, you must first install the following dependences:

* biorefineries: 2.32.2
* biosteam : 2.52.20
* thermosteam: 0.52.18
* thermo: 0.6.0
* fluids: 1.3.0
* chemicals: 1.5.0
* flexsolve: 0.5.10
