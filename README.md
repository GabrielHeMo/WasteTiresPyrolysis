# WasteTiresPyrolysis
This repository contains the source code for the techno-economic, life cycle, and safety analyses for the pyrolysis of waste tires as discussed in [[1]](#1). 

## Getting started

A tutorial on how to rerun the analyses can be found the Pyrolisis_Biosteam_Jupyter.ipynb notebook. In order to run the code, you must first install the following dependences in Python 3.11:

* biorefineries: 2.32.2
* biosteam : 2.52.20
* thermosteam: 0.52.18
* thermo: 0.6.0
* fluids: 1.3.0
* chemicals: 1.5.0
* flexsolve: 0.5.10
* pyomo: 6.10.0
* ipopt: 1.0.3

You can install the pyrolysis package with all its dependencies by running "pip install ." within your termial/command line. Make sure that your directory is the repository; it should contain the setup.py file and the pyrolysis folder.
