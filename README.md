# WasteTiresPyrolysis
This repository contains the source code for the techno-economic, life cycle, and safety analyses for the pyrolysis of waste tires as discussed in [[1]](#1). 

## Getting started

A tutorial on how to rerun the analyses can be found the Pyrolisis_Biosteam_Jupyter.ipynb notebook.
